• git init (инициализация нового репозитория)
  • touch (создать файл)
  • git status (показать статус файлов какие записаны а какие нет)
  • git add . (добавить всё в гит)
  • git commit -m "" (создание точки сохранения)
  • git branch (показывает все ветки проекта)
  • git checkout -b "" (создать новую ветку)
  • git checkout "" (переключиться на ветку)
  • git merge "" (соединить ветки)
  • git diff (пишет какие изменения были в файлах) :q (выйти из режима)
  • cat "" (вывести конкретный файл почитать)
  • git branch -d "" (удаляет локальную ветку)
  • git -- hard (удаляет обновления до последнего комита)
  • git branch -m "" (создать главную ветку на гитхабе)
  • git push -u origin "" (Запушить/отправить код в удалённый репозиторий)
  • git pull origin ""(вытаскивает изменения из репозитория на комп)
  • git clone "ссылка из гитхаба" (копирует проект из гитхаба)
